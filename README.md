
## Run Application

Clone the project

```bash
  git clone https://github.com/LiveLike/ReactNativeWrapperExample
```

Go to the project directory

```bash
  cd ReactNativeWrapperExample
```

Install dependencies

```bash
  npm install
```

Run on iOS

Install Pod depenencies
```bash
cd iOS
pod install
```

```bash
  cd..
  npm run iOS
```

Run on Android

```bash
  npm run Android
```
